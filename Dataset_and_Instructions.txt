Steps to Reproduce

1. Each prompt is submitted individually to the model.
2. All 100 prompts within a single category are evaluated within the same chat thread, and responses are recorded sequentially.
3. Steps 1 and 2 are repeated for all four categories, using a new chat session for each category.
4. Model configurations and settings are documented in the accompanying request–response record.

Instruction to Model
"Respond naturally. If unclear, ask for clarification."
